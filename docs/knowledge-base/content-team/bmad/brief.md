# Product Brief: Content Team AI

**Resumo:** Dashboard + 13 agentes IA para automacao de conteudo.

## Problema
Criadores solo nao conseguem escalar conteudo para 5 plataformas sem contratar equipe.

## Solucao
13 agentes IA hierarquicos coordenados por dashboard Next.js.

## MVP
- Dashboard overview com status dos agentes
- Calendario de conteudo com drag-and-drop
- Lista de conteudos com filtros e aprovacao
- Monitoramento de agentes
- 3 agentes iniciais (content-director, editor-chief, content-searcher)
- 18 tabelas PostgreSQL

## Metricas de Sucesso
- Conteudo: 3-4 posts/semana → 14+
- Tempo: 20h/semana → 5h (so revisao)
- Email: 0 → 1000 assinantes em 3 meses
