# Web Search - Pesquisa Web

Pesquisa na web usando as ferramentas built-in do Claude Code: WebSearch e WebFetch.

## Como Usar

O Claude Code tem ferramentas nativas de busca:
- **WebSearch** — pesquisar na web por termos
- **WebFetch** — buscar conteudo de uma URL especifica

Nao precisa de API key nem curl. Use diretamente.

## Quando Usar

- Pesquisar tendencias de conteudo
- Buscar informacoes atualizadas
- Verificar fatos e dados
- Encontrar inspiracao para posts

## Regras

- Sempre apresentar fontes das informacoes
- Resumir conteudo em PT-BR
- Citar URLs relevantes
